# Mon Calendrier Sportif — GitHub Pages

## Étape 1 — Mettre ta clé dans le code
1. Ouvre `index.html` avec le Bloc-notes (Windows) ou TextEdit (Mac)
2. Cherche la ligne : `var GEMINI_API_KEY = 'VOTRE_CLE_ICI';`
3. Remplace `VOTRE_CLE_ICI` par ta clé `AIza...`
4. Sauvegarde

## Étape 2 — Mettre en ligne sur GitHub Pages
1. Va sur **github.com** → connecte-toi
2. Ouvre ton repo `sportcal`
3. Clique **Add file** → **Upload files**
4. Glisse `index.html` et `manifest.json` (remplace les anciens)
5. Clique **Commit changes**
6. Va dans **Settings** → **Pages** → Source: **Deploy from branch** → branch: **main** → **Save**
7. Ton URL : `https://TON_USERNAME.github.io/sportcal`

## Étape 3 — Installer sur iPhone
Safari → ouvre l'URL → Partager ↑ → Sur l'écran d'accueil
